# Архитектура баз данных для проекта SFMShop

## Общая схема

- **PostgreSQL**: Основные данные (users, products, orders)
- **Redis**: Кэш и сессии
- **MongoDB**: Логи и аналитика

## Обоснование выбора

### PostgreSQL
- Структурированные данные
- ACID транзакции
- Связи между данными

### Redis
- Высокая скорость
- TTL для автоматического удаления
- Временные данные

### MongoDB
- Гибкая структура
- Масштабируемость
- Аналитические запросы
